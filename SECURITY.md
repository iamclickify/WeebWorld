# Security Policy

## Reporting a Vulnerability

Thank you for taking the time to responsibly disclose security issues for this project.

If you believe you have found a security vulnerability in this repository, please follow these steps:

1. If possible, use GitHub's private security advisories feature: https://github.com/orgs/iamclickify/security/advisories (or repository Security -> Advisories).
2. If you cannot use the advisory feature, open an issue with the label `security` and add as much detail as you can. If you prefer a private disclosure, please email the project maintainer via the GitHub profile: https://github.com/iamclickify

What to include:
- Steps to reproduce the issue
- Expected vs actual behavior
- Any proof-of-concept code or PoC URL
- A suggested mitigation if you have one

We will respond within 5 business days.

---

No guarantees are implied by this policy; we will do our best to evaluate and respond. Thank you for helping keep this project safe.
